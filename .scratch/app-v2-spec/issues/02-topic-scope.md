# ขอบเขตหัวข้อเรียน: เปิดอิสระแค่ไหน

Type: grilling
Status: resolved

## Question

ผู้ใช้เลือกเรียน "อะไรก็ได้" โดยให้ AI generate roadmap สด, หรือมีหัวข้อ curated คุณภาพสูง (แบบ ML roadmap เดิมที่มีลิงก์ Kaggle จริง) เป็นแกน + เปิดอิสระเป็นตัวเสริม? ถ้ามี curated ควรมีกี่หัวข้อ อะไรบ้าง? และ roadmap ที่ AI generate ต้องมี guardrail อะไร (เช่น ห้ามมั่วลิงก์ — บทเรียนจาก ml-quest ที่ต้อง pin URL จริง)

## Answer

ตัดสินใจร่วมกับเจ้าของ 12 ก.ค. 2026:

1. **โครงสร้าง: Curated เป็นแกน + พิมพ์อิสระได้** — หน้าเลือกหัวข้อโชว์หัวข้อคัดสรรเด่น ๆ พร้อมช่อง "พิมพ์หัวข้อที่อยากเรียนเอง" ให้ AI วาง roadmap
2. **หัวข้อ curated เปิดตัว 6 หัวข้อ** (สายเทค + ทักษะทำงาน/ชีวิต):
   - Python เริ่มจากศูนย์
   - Data/ML — ยก roadmap Kaggle จาก ml-quest เดิมมาใช้ ประหยัดเวลาคัดสรร
   - สร้างเว็บ (HTML/CSS/JS)
   - ใช้ AI ให้เป็น (prompt/เครื่องมือ AI ในการทำงาน)
   - Excel/Google Sheets
   - การเงินส่วนบุคคล (มีแหล่งไทยเยอะ เช่น ตลาดหลักทรัพย์/SET e-Learning)
3. **ภาษาแหล่งบทเรียน: ไทยก่อน อังกฤษเสริม** — เควสหาแหล่งไทยเป็นอันดับแรก (YouTube ไทย, บทความไทย, คอร์สฟรีไทย) ถ้าไม่มีค่อยใช้อังกฤษ ตัวเควส/คำอธิบาย/Quest Coach เป็นไทยเสมอ
4. **Guardrail โหมดพิมพ์อิสระ: ลิงก์ได้เฉพาะแบบปลอดภัย** — AI เขียนเนื้อหาเควสเองเต็มที่ แต่ลิงก์ที่แนบได้มีแค่ 2 แบบ: (ก) หน้าหลักของโดเมนใน whitelist (ข) ลิงก์ค้นหา เช่น YouTube search URL พร้อมคำค้น — ห้ามแต่ง deep URL เองเด็ดขาด (บทเรียนจาก ml-quest ที่ Gemini แต่ง URL ไม่มีจริง)

**สิ่งที่ตามมา:** งานคัดแหล่งบทเรียนไทยจริงต่อหัวข้อ + ทำ whitelist โดเมน → แตกเป็น ticket ใหม่ "คัดแหล่งบทเรียนไทยสำหรับ 6 หัวข้อเปิดตัว" (12)
